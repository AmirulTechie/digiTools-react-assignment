# DigiTools - Digital Tools Buying Website

## 📌 Project Overview

DigiTools is a modern single-page web application (SPA) built with React that allows users to explore and purchase various digital tools. The platform provides a smooth user experience with dynamic product listing, cart management, and interactive UI elements.

---

## 🧠 Description

This project simulates a digital marketplace where users can browse different tools, view their features, and add them to a cart. It demonstrates core React concepts like component-based architecture, state management, and dynamic rendering, along with a clean and responsive UI design.

---

## ⚙️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🌼 DaisyUI
* 🧠 JavaScript (ES6+)
* 🔔 React-Toastify
* 📦 JSON (for product data)

---

## ✨ Key Features

### 🛒 1. Dynamic Cart System

* Add products to cart
* Remove individual items
* Clear entire cart with "Proceed to Checkout"
* Live cart count update in navbar

---

### 🔄 2. Toggle View System

* Switch between **Products** and **Cart** instantly
* No page reload (SPA behavior)
* Default view set to Products

---

### 🔔 3. Interactive Notifications

* Toast alerts for:

  * Adding to cart
  * Removing items
  * Checkout actions
* Improves user experience and feedback
